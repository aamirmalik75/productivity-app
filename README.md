## Installation

1. Clone the repository
```bash
git clone https://github.com/aamirmalik75/productivity-app.git
```

2. Frontend setup
```bash
cd frontend
npm install
npm start
```

3. Backend setup
```bash
cd backend
composer install
php artisan migrate
php artisan serve
```
